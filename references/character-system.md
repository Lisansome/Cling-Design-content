# Cling Design Character System

## 角色策略

这套 skill 默认要有一个主角色。

不是小黑，不是冷剪影，也不是吉祥物。

优先角色类型：

1. 美式辣妹 / poster girl 气质角色
2. 二次元与时尚插画之间的混合角色
3. 有攻击性、漂亮、聪明、略不耐烦的年轻女性角色
4. 一个看起来“知道自己很懂，也懒得解释太多”的角色

## recurring archetype

如果用户没有另外给出新人物设定，默认使用同一个 recurring archetype：

- 短 bob 头，长度到下巴附近，发尾内扣，轮廓整齐
- 圆形或略 oversized 的深色墨镜
- 黑色高领上衣
- 小而冷的脸
- 薄嘴唇
- 不要明显长睫毛
- 不要大耳环
- 不要频繁换成别的发型、饰品、脸型
- 气质是：冷静、拽、知道自己在干嘛、不需要讨好谁

一句话：

- **人物是 recurring family，不是 strict 固定资产。**

## 角色应该长什么样

优先：

- 头肩或半身为主，尽量近
- 眼神直接、挑眉、无语、轻蔑、若有所思、带点拽
- 以 recurring family 为准，只在不破坏识别度的前提下做姿态变化
- 墨镜、高领、短 bob 头优先保留
- 角色要像主角，不像装饰

不要：

- 夸张卖萌大眼
- 幼儿比例
- 软妹甜妹感
- 纯头像商品立绘
- 过度裸露或色情化
- 换成长发、卷发、复杂耳饰、浓睫毛、厚嘴唇

## 角色职责

角色是整张图的情绪中枢。

她不是来讲步骤的，而是来承受这些内容的情绪：

- 认真过头
- 想讲清楚
- 被打断
- 无奈
- 思考
- 整理
- 终于理顺
- 以及“别把我当傻子”的态度

## 角色动作

优先动作：

- 抱臂
- 托腮但带不耐烦
- 手里夹着纸或手机
- 被信息围住但神情冷静
- 抬眼斜看
- 手指勾着一张纸
- 被问题气泡怼脸
- 倚着桌子或卡片站着

动作可以不大，但气场要准。

如果一张图里需要出现多个小人：

- 他们应该像**同一类人被缩小后重复出现**
- 不是“同风格不同女孩”
- 但不要求每个小人都像 strict 资产一样完全一致

## 周边元素

围绕角色的元素可以有：

- 纸张
- 便签
- 气泡
- 信息卡
- 感叹号
- 括号
- 小点点
- 简单箭头尾巴

这些元素的职责是：

- 让角色脑内状态可视化
- 让内容像漂浮在身边

不是：

- 把画面做成流程图模板

## 禁忌

不要：

- 让角色变成教程老师
- 角色比信息还少，像装饰
- 过度萌化
- 过度写实
- 拍成冷杂志人像
- 太乖太温柔
- 小人一缩小就丢掉墨镜、bob 头和黑高领
- 为了适配流程图而把人物重发明成别的人

这套风格要有吸引力，但不是讨好感。
