# Cling Design Style DNA

## 核心气质

这套风格要的是：

- 暖
- 拽
- 干净
- 有角色
- 有一点怪
- 有流行海报感
- 美式辣妹但不低俗

第一眼应该像：

- 角色感很强的社媒正文图
- Y2K / teen magazine / poster girl 气质内容图
- 带一点手绘粉感的辣妹信息海报

而不是：

- 冷海报
- 杂志封面模板
- 正式信息图
- SaaS 插画
- 软萌手帐贴纸图

## 背景与材料

默认背景：

- 奶油白 / 暖米白
- 轻微纸张或粉笔纹理
- 直角画幅，像印刷海报或杂志内页
- 不要圆角底板，不要卡片感

材料感：

- 粉蜡笔
- 彩铅
- 粗颗粒数码画笔
- 手绘纸面
- 轻微复古印刷颗粒

不要：

- 玻璃感
- 发光
- 金属质感
- 油亮 3D
- 太真实的纸张摄影纹理

## 色板

主色：

- 奶油白 / warm cream
- 黑色 / 柔黑

辅助色：

- 青绿色 / teal
- 橙色 / soft orange
- 芥末黄 / mustard

规则：

- 以黑线和奶油底为基底
- teal 和 orange 是最常用的两块点色
- 黄只做小面积提醒或节奏点
- 不要大面积冷蓝紫
- 不要高饱和荧光色

## 线条与质感

优先：

- 粗细不完全均匀的黑色手绘线
- 有一点毛边和粉感
- 不是技术图纸线条

不要：

- 极细硬核几何线
- 精准矢量 UI 轮廓
- 复杂阴影

## 文字策略

可以有字，但像“贴在角色周围的小信息”。

适合：

- 短句
- 词组
- 气泡字
- 小纸条
- 小主标题

不适合：

- 大段解释
- 标准标题 + 副标题海报结构
- 逻辑分栏
- 软萌手账注释

补充：

- 这套 skill 里的字更像少量信息贴片或一个小主标题，不要抢场景

## 画面密度

中心一定有一个主角色。

周围可以有：

- 纸张
- 气泡
- 摘要卡片
- 小符号
- 情绪括号、感叹线、箭头尾巴

但总原则：

- 主角色先被看见
- 周围信息只做陪衬
- 不要把角色淹掉
- 信息碎片更像“被贴上去的舆论和工作噪音”，不是可爱装饰

## 禁止项

明确禁止：

- 科技蓝紫
- 玻璃拟态
- 冷灰企业插画
- 纯流程图模板
- 太萌的 Q 版吉祥物
- 低幼儿童绘本感
- 太满的手帐贴纸爆炸风
- 圆角卡片 UI 感
- 乖巧软妹感

## 成图判断

如果第一眼更像：

- 一个很会想事的角色
- 被内容碎片围绕
- 有点难搞、有点漂亮、有点烦
- 像一张会被存下来的“辣妹内容图”

就对了。

如果第一眼更像：

- 教程模板
- 海报模板
- 企业插画

就错了。
