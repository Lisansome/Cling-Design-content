# Cling-Design-content

中文内容正文配图 skill。

这套 skill 现在只做一件事：

- 生成暖奶油底、粉蜡笔纹理、角色主导、轻信息贴片感的中文正文配图、流程图、情绪图与观点插图

不再包含 strict 封面路线。当前版本是 **approx-only**。

## 适合的图

- 正文配图
- 流程图
- 情绪图
- 观点插图
- 工作状态图

## 不适合的图

- 固定人物封面
- 一句话大字钩子封面
- 头像延展
- strict 人设封面

## 使用原则

- 先让意思成立，再让气质成立
- 人物保同一类气质，不保像素级同一个人
- 流程图优先场景和动作，不要做成低配框箭头模板
- 默认比例：
  - `16:9` 用于流程/结构/横向表达
  - `3:4` 用于正文卡图
